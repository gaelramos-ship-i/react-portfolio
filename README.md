# Portfolio

This portfolio was created as part of my learning journey at **La Plateforme** to improve my skills in **React** and showcase my projects during my internship search.

The goal of this project was to build a modern and responsive personal website while learning the fundamentals of React development.

---

## Technologies Used

- React
- Typescript
- SCSS

## Getting Started

Start the development server:

```bash
npm run dev
```

## Learning Objectives

This project helped improve skills in:
- React fundamentals
- component architecture
- props and state management
- responsive front-end development
- project organization
- portfolio presentation
